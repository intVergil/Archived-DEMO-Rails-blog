# vBlog

学习使用ruby on rails，编写个人主页。

### 生产环境123

ruby：2.5.1
rails：5.2.0
- 前端框架
  - bulma
  - fontawesome
  - react.js
- 后端框架
- 数据库
  - MySQL
- 开源托管
  - github
- 服务器
  - AWS

### 板块说明

- 主页：静态的总体概览页面。
- 博客：动态完整的博客系统（/admin 后台登陆）。
- 摄影：动态的图片上传系统（点赞系统）。
- 设计：学习使用多种静态设计效果。
- 关于：个人简历及联系方式。
